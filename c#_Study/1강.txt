프로그램의 시작은 메인함수다.
메인함수가 시작되면서 프로그램이 시작된다.

using 다음 특정한 namespase를 사용해서 시스템의 보따리를 사용
사용하고싶은 툴을 사용한다.
Ex) System.Collections.Generic
	시스템안에 컬렉션안에 제너릭이라는걸 사용하는것
	보따리안에 보따리안에 보따리
	사용하고싶은 것을 정리를 잘 해놓은 것

namespace는 내가 보따리를 만드는 것
Class 는 내가 툴을 만드는 것


internal 키워드는 C# 프로그래밍 언어에서 접근 제어 지시자(access modifier) 중 하나
internal로 지정된 클래스나 멤버는 같은 프로젝트 또는 어셈블리 내에서만 사용되도록 제한됩니다. 
다른 어셈블리에서는 해당 멤버에 접근할 수 없습니다.

Console.ReadKey();  //키를 받을때까지 기다린다.


메인함수에서의 String[] args는
뭔가를 받을수도있기 때문에 써준다

예를들어 cmd창에서 Hello 파일이 있다
Hello aaa 를 쓴다면 메인함수에서 aaa를 인자로 인식해서 받아 사용할수있다.
그래서 String[] args는 그때 사용된다.
